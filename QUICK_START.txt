📋 SETUP (First Time Only)
─────────────────────────────────────────────────────────────

1. Install Python Dependencies:
   pip install -r requirements.txt

2. Download Dataset from Kaggle:
   Option A (Recommended): .\setup.ps1
   Option B (Manual):      python download_data.py
   Option C (Web):         https://www.kaggle.com/datasets/aravind100k/20m-combined-db

3. Test Installation:
   python main.py eda

🚀 QUICK COMMANDS
─────────────────────────────────────────────────────────────

Interactive Recommender:
   python main.py recommend

Exploratory Data Analysis:
   python main.py eda

Hidden Gems (All):
   python main.py hidden-gems

Hidden Gems (By Genre):
   python main.py hidden-gems "Sci-Fi"
   python main.py hidden-gems "Comedy"
   python main.py hidden-gems "Drama"
